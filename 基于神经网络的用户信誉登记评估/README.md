# 主题一，方法一

## 文件夹结构

* code： 存放代码 jupyter notebook形式

* code_pdf： 存放jupyter notebook 的 pdf形式

* data ：存放训练模型所使用的数据

* 训练好的模型： 存放训练好的完整模型

  > 使用 torch.load()即可加载

* 训练结果：存放没有给出level的用户的星级

## 如何运行

安装anaconda，构建Python环境 3.10.4，之后pip安装code文件夹中 requirements.txt中的依赖。

在anaconda命令行中输入jupyter-lab即可打开
